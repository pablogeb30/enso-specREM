OBJ-0001 Sistema de registro de gastos
Versión 1.0 ( 05/12/2023 )
Autores
  • Aarón García Filgueira
  • Pablo García Fuentes
Fuentes
  • José Ángel Taboada Gonzalez
Descripción El sistema deberá permitir el registro de un gasto configurando sus particicpantes, calculandoa automáticamente el balance y registrando una fecha asociada a la operación. También ofrecerá la posibilidad de adjuntar un justificante y/o asociar el gasto a una actividad existinte en el sistema. Para cada gasto se pueden consultar saldos y deudas; y registrar nuevos pagos de deuda.
Subobjetivos Ninguno

Versión 1.0 ( 12/12/2023 )
Autores
  • Aarón García Filgueira
  • Pablo García Fuentes
Fuentes
  • José Ángel Taboada Gonzalez
Descripción El sistema deberá permitir la creación y eliminación de grupos, la inclusión y eliminación de miembros y la gestión de anfitriones. Existirá también la opción de bloquear usuarios o restringir su participación en algunas actividades.

Cada grupo deberá integrar un chat que maneje envio de mensajes y sus respectivas notificaciones.

OBJ-0004 Sistema de gestión de actividades
Versión 1.0 ( 12/12/2023 )
Autores
  • Aarón García Filgueira
  • Pablo García Fuentes
Fuentes
  • José Ángel Taboada Gonzalez
Descripción El sistema deberá permitir la creación y programación de actividades para una fecha determinada que se indicará en el calendario.
Sobre las actividades se podrán publicar reseñas, esetablecer una puntuación y consultar las opiniones que hayan publicado otros usuarios.
Subobjetivos
  • [OBJ-0005] Sistema de recomendaciones autamáticas.: El sistema deberá recomendar a los usuarios de manera activa actividades en función de sus valoraciones y de las preferencias prestablecidas por los administradores.
